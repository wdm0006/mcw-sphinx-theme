## [0.1.0] - 2025-12-25

### Added

- Initial release of the Wabi Sphinx Theme
- Matching design with the Hugo Wabi theme (mcginniscommawill.com)
- Warm color palette: cream (#faf8f5), sand, terracotta (#b8574c)
- Typography: Fraunces (headlines), Newsreader (body), Instrument Sans (UI), IBM Plex Mono (code)
- Custom Pygments syntax highlighting style (WabiStyle)
- Unified navigation header matching the main website
- Responsive design with mobile-friendly collapsible navigation
- Breadcrumb navigation support
- Full Sphinx compatibility (extends basic theme)
- Comprehensive documentation and examples

### Technical

- Python 3.9+ support
- Sphinx 7.0.0+ compatibility
- pip/uv installable package
- Pre-commit hooks for code quality
- GitHub Actions CI/CD workflows
- pytest test suite with 80%+ coverage target
