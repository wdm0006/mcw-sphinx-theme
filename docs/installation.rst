Installation
============

Requirements
------------

- Python 3.9 or higher
- Sphinx 7.0.0 or higher

Install from PyPI
-----------------

The recommended way to install the Wabi Sphinx Theme is via pip::

    pip install wabi-sphinx-theme

Or if you're using uv::

    uv add wabi-sphinx-theme

Install from Source
-------------------

You can also install directly from the GitHub repository::

    pip install git+https://github.com/wdm0006/wabi-sphinx-theme.git

Or clone and install in development mode::

    git clone https://github.com/wdm0006/wabi-sphinx-theme.git
    cd wabi-sphinx-theme
    pip install -e .

Verify Installation
-------------------

You can verify the installation by checking the version::

    python -c "import wabi_sphinx_theme; print(wabi_sphinx_theme.__version__)"

This should print the current version number.
